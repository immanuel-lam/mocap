# Impeccable Design Context — MoCap

## Design Context

### Users
Small VFX and animation studios. A DOP or TD who is capturing real camera motion for integration into Blender/Maya/Houdini. Professional context — the output feeds into client deliverables. The UI needs to read as credible and capable to collaborators who will see it over someone's shoulder. Speed matters; this is used during shoots, not just at a desk.

### Brand Personality
**Precise. Cinematic. Instrumental.**

Not "cool developer tool" — that's the lazy reflex. The right reference is professional cinematography and post-production equipment: ARRI camera menus, Blackmagic Design UI, DaVinci Resolve's grading panels. The personality of a very good piece of specialized equipment that does one thing with total confidence.

### Aesthetic Direction
- **Theme**: Dark — this is used in grading suites, dark offices, on-set. Eye strain on a light UI is unacceptable for long sessions.
- **Color palette**: Near-black UI surfaces (#0e0e12, #141418), very tinted toward a cold blue-grey. Single accent color used sparingly — warm amber (#e07c2a or similar) for active/recording states, like timecode displays on professional cameras. Green only for tracking-quality confirmation, not as a brand color.
- **Typography**: Condensed label font (Barlow Condensed — already present, appropriate) for UI chrome. A clean, slightly wide grotesque (not Inter or DM Sans) for data readouts. Monospace only for numerical values, not for general UI text.
- **Density**: Data-rich but organized. Think NLE panel, not consumer app. Small text is fine — professionals have screens and eyeballs.
- **3D viewport**: The centerpiece. Panels defer to it. UI chrome should visually recede.

### Design Principles
1. **The viewport is the product** — every UI decision should make the 3D view feel more prominent, not less.
2. **Color communicates state, not decoration** — amber = active/recording, green = tracking normal, red = error/loss. No color for brand identity.
3. **Density over whitespace** — professional tools are dense. Whitespace signals consumer product; tightly organized panels signal professional instrument.
4. **Restraint is the aesthetic** — no gradients, glows, or decorative animation. Motion only for functional state changes (playback, recording transitions).
5. **Text is data** — all text is either a label, a value, or an action. No copy, no explanation, no marketing language.
